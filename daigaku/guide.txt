아직 서버와 데이터베이스 서버가 없기 때문에 로컬에서 테스트 동작 확인을 위한 방법

테스트를 위한 초기 세팅법 ( 비주얼 스튜디오 코드 , 파이썬 )

1 . 위의 서버 코드를 동작시키기 위한 모듈 전부 다운 받으세요 !
2 . 코드에 주석으로 달려있는 설명을 보고 로컬 데이터베이스와 연결해주세요 !
3 . API 키는 그대로 둬도 괜찮습니다 ,  본인 API 키를 입력해도 무관합니다
 // 데이터베이스는 now_db 의 가장 처음 수정 전 버전을 사용하였습니다 ( 추후 교수님 서버에 업데이트 하면서 변경 예정 )
4 . 데이터베이스 heidiSQL 과 MariaDB를 사용해 now_db 작성해주세요 ( 이름은 같게 , 만약 다르다면 코드에서 DB 이름과 같게 수정하기 )
5 . 이제 비쥬얼 스튜디오 코드에서 디버깅 해주세요 ( 아무 에러도 안뜨고 디버깅 완료 되어야 합니다 )
6 . 작성한 파이썬 코드가 존재하는 폴더에서 빈 곳 우클릭 -> 터미널 열어주기
7 . 터미널 오픈후 -m uvicorn serverprpr:app --reload 입력 ( ERROR 가 뜨지 않는다면 로컬 서버 오픈 성공 )
- 만약 되지 않는다면 python -m uvicorn serverprpr:app --reload 을 입력해보세요

위의 과정을 모두 마쳤다면 아래의 과정 따라하세요

1 . http://127.0.0.1:8000/docs 접속
2 . GET / Read Root 를 클릭 후 오른쪽 위에 있는 Try it out 버튼을 찾아 클릭
3 . Excute 클릭 후 아래 Responses 에 Code 200 과 함께 Details 에 정상적으로 실행중입니다 ! 라는 메시지가 보이는지 확인해주세요
- 되지 않는다면 과제 단톡방에 문의 부탁드립니다 !
4 . Code 200 까지 정상적으로 보인다면 아래의 과정 따라해주세요
